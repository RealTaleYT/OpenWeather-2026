## Oper Wetather web application
# ES
Bienvenidos a mi página aplicación web donde podréis revisar el clima de una ciudad o de tu ubicación actual.\
También podéis cambiar el idioma de los datos del clima en la parte de arriba a la derecha de la pantalla.
Para arrancar la versión web, se necesita Ionic:
```bash
npm install -g @ionic/cli
```
Una vez instalado ionic y el repositorio, se tiene que poner el siguiente comando en la terminal dentro de la carpeta del proyecto:
```bash
ionic serve
```
Si quieres usar la versión móvil, se necesita android studio([https://developer.android.com/studio?hl=es-419](https://developer.android.com/studio?hl=es-419)) y un teléfono (ya sea físico o emulado),
Una vez lo tengas, utiliza el siguiente comando en la terminal dentro del proyecto: 
```bash
ionic cap run android
```
Y por último, darle a arrancar.

```bash

```

```bash

```

once u have this, use the command:
